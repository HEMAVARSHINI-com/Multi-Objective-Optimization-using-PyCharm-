# Multi-Objective-Optimization-using-PyCharm-

Graphene Nanostructure Field-Effect Transistors (GNSFETs) are promising platforms for highly sensitive biosensing, offering rapid detection with low limits. Optimizing their performance requires balancing multiple conflicting objectives, including sensitivity, response time, power consumption, false positive rate, and long-term stability. In this work, we propose a machine learning–assisted multi-objective optimization framework for GNSFET biosensors using the Non-dominated Sorting Genetic Algorithm III (NSGA-III). Key decision variables, including gate bias, sampling rate, amplifier gain, functionalization density, and filter characteristics, are systematically tuned to explore trade-offs across five objectives. Simulation results demonstrate that the framework generates robust Pareto fronts, enabling selection of balanced operating points suitable for diverse applications, from rapid emergency detection to high-accuracy clinical screening. This approach offers a scalable pathway for resource-efficient, real-time deployment of GNSFET-based biosensors in biomedical and environmental monitoring scenarios.
